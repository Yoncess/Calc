Техническое задание на разработку консольного приложение «Калькулятор»
Цель работы: разработать консольное приложение «Калькулятор» на языке C#, предназначенного для выполнения основных арифметических операций над двумя числами. Программа должна обеспечивать корректный ввод данных пользователем, выбор операции и вывод результата вычисления. После выполнения операции программа выводит результат на экран. В случае некорректного ввода программа должна уведомить пользователя об ошибке и завершить выполнение без сбоев.
Функциональные возможности: 
Программа должна реализовывать следующие арифметические действия: 
1.	Сложение (+)
2.	Вычитание (-)
3.	Умножение (*)
4.	Деление (/)
5.	Возведение в степень (^)
Также программа должна включать проверку корректности вводимых данных: проверка на нечисловые значения, проверка на правильный выбор арифметической операции, ошибка при делении на ноль.
